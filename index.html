Project Overview

EcoNoc Glasgow provides a comprehensive analytics dashboard for monitoring bat activity and light pollution impact across Glasgow's parks. Our system tracks real-time bat populations, analyzes light intensity patterns, and supports conservation efforts through data-driven insights and community engagement.

Bat Activity Monitoring

Real-time bat count, species detection, and activity pattern analysis with interactive species cards

Light Pollution Tracking

Live light intensity readings, sensor network mapping, and spectral composition analysis

Impact Analysis

Correlation charts, event impact comparison, and policy recommendations for Glasgow City Council
